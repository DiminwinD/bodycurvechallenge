# BodyCurveApp

## Description
BodyCurveApp is a fitness transformation platform featuring the BodyCurve Challenge and BodyCurve Control.

## Main Features
- Landing page with email capture
- Dashboard for user progress tracking
- PDF guide delivery
- Payment integration
- Future AI coach functionality

## Project Structure
- frontend/: Frontend components including landing page, dashboard, and challenge pages
- backend/: Backend services for authentication, payments, and progress tracking
- database/: Database schemas and migrations for users, workouts, and progress
- ai/: AI-powered coaching and motivation engine
- assets/: Static assets like images and documents
- docs/: Documentation files
- infrastructure/: Infrastructure as code and deployment scripts
- scripts/: Utility scripts for development and deployment
- tests/: Test suites for the application

## Development Rules
- Keep code organized and modular
- Work only inside this project folder
- Follow best practices for security and performance
